# Shinji Ota Portfolio

This is the personal portfolio website for Shinji Ota, Safety Culture Translator.

## Structure
- `index.html`: Main entry point.
- `style.css`: Stylesheet with custom properties and animations.
- `script.js`: Interactive elements (cursor, scroll animations).
- `assets/`: Contains generated images.

## Deployment
1. Push this repository to GitHub.
2. Go to **Settings** > **Pages**.
3. Select the **main** branch and `/root` (or `/sota_hp` if that's the root of the repo) folder.
4. Save.

## Design
- **Theme**: Urban, Sophisticated, Dark Mode.
- **Fonts**: Outfit (Headings), Inter (Body).
- **Features**: Glassmorphism, Smooth Scroll, Custom Cursor.
